Your tool for Digital Scholarly Editions

Edirom Online is an open-source web application for publishing digital editions of music. Explore manuscripts, transcriptions, scholarly commentary, and more in an interactive environment.

View on GitHub

Key Features

Standards-Based

Supports TEI and MEI, ensuring interoperability and long-term sustainability of your scholarly data.

IIIF Integration

Display high-resolution manuscript and print images with full IIIF support for seamless image delivery.

Flexible Architecture

Modular design for customizable workflows and future feature extensions tailored to your edition.

Dynamic Views

Combine facsimiles, transcriptions, commentary, and more in interactive and user-friendly layouts.

Open Source

Freely available under an open license with an active community driving ongoing development.

Community Driven

Developed and maintained by scholars and developers in the digital humanities community.

Install Edirom Online

The local setup of an Edirom Online instance - fetching and building of frontend and backend and inclusion of edition packages - can be done with a few docker compose commands which are documented in the README of the github repository.
