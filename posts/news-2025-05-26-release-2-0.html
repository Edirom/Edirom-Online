Release of Edirom Online v2.0.0

2025-05-26

With separate software bases for backend and frontend and overhauled build process and documentation this is quite a big conceptual change to the version 1.0.0 which was the last monolithic Edirom Online version.

The new version makes it easier to setup an own Edirom instance - only Docker is needed and no more manual building with ant or build scripts necessary. Local and remote xar packages for editions can be provided during the starting of the Docker environment.

The actual software development continues in the backend and frontend repositories.
