Release of Edirom Online API v1.0.0

2025-06-18

We are excited to announce the release of the Edirom Online API v1.0.0!

The API provides a structured and transparent view into the backend of Edirom Online, enabling developers to interact with the platform programmatically. It is available in OpenAPI v3.0.1 format and can be accessed through the Swagger UI.

Shout out to the amazing Edirom Online community for making this possible.

Git Repository https://github.com/Edirom/Edirom-Online-API API in Swagger UI: https://edirom.github.io/Edirom-Online-API/#/
API Release on Zenodo: https://zenodo.org/records/15690219
